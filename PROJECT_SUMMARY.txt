PROJECT OVERVIEW
================================================================================
CoverQuest is a 32-player NFL pick'em game where users:
  • Pick 6 NFL games against the spread each week
  • Apply Game Changers (Castle Wall 🛡️, Let It Ride 🚀)
  • Accumulate points based on coverage accuracy
  • Compete across 18-week season for best total score
  • Compete within divisions and conferences

DATABASE SCHEMA (8 tables)
================================================================================

users (32 total)
├─ id (UUID, primary key)
├─ email, username (unique)
├─ password_hash (SHA256)
├─ avatar_color (hex)
├─ division_id (FK → divisions)
├─ conference_id (FK → conferences)
├─ is_admin (boolean)
├─ created_at, updated_at
└─ Indexes: email, division_id, conference_id

games (~288 for 18 weeks)
├─ id (ESPN ID, primary key)
├─ week (1-18)
├─ away_team, home_team, spread
├─ away_abbr, home_abbr
├─ favorite_team
├─ kickoff_time (UTC)
├─ away_final_score, home_final_score
├─ status (scheduled/live/final)
├─ synced_at (last ESPN update)
└─ Indexes: week, status

picks (~3,456 for season)
├─ id (UUID, primary key)
├─ user_id (FK → users)
├─ week (1-18)
├─ game_id (FK → games)
├─ picked_team
├─ game_changers (JSON)
├─ submitted_at
└─ Indexes: user_id+week, user_id+game_id

scores (calculated after games)
├─ id (UUID, primary key)
├─ user_id (FK → users)
├─ week (1-18)
├─ game_id (FK → games)
├─ picked_team
├─ actual_differential
├─ game_changers (JSON)
├─ game_changer_multiplier (1.0 or 1.5)
├─ floor_applied (boolean)
├─ final_score
├─ calculated_at
└─ Indexes: user_id+week

season_stats (32 total, updated weekly)
├─ user_id (FK → users, unique)
├─ total_score
├─ total_weeks_played
├─ division_rank (1-4)
├─ conference_rank (1-16)
├─ overall_rank (1-32)
├─ best_week (1-18)
├─ best_week_score
├─ calculated_at
└─ Indexes: total_score DESC

divisions (8 total)
├─ id (AFCE, AFCN, AFCS, AFCW, NFCE, NFCN, NFCS, NFCW)
├─ name (string)
└─ conference_id (FK → conferences)

conferences (2 total)
├─ id (AFC, NFC)
└─ name (string)

week_locks (1 per locked week)
├─ week (1-18, unique)
├─ locked_at
└─ locked_by_admin_id (FK → users)

audit_logs (all admin actions)
├─ id (UUID)
├─ admin_id (FK → users)
├─ action (string)
├─ details (JSON)
└─ created_at
